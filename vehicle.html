MVC Vehicle Renting System

Description

This app allows users to sign into the system, view and rent vehicles. There is an admin area that allows the admin to manage the vehicles and any changes made will be reflected for all users to see. Only the admin can make the change to return the vehicle to the system.

Tech Stack: Java, Java GUI
