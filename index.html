Inventory System for Products

This System is used to hold an inventory of products and their ingredients. The products can be requested and assigned to distributors who will then go and sell the products.Ingredients can be added and a product will consist of different ingredients. There is an analysis section which consists of different charts to let the user know about what is taking place on the app. Like which distributor has requested the most products or how many units of each product is available.
Made With bubble.io

Clipboard App

This app allows a user to copy a piece of code used to make different components in tailwindcss for a react app such as a login form, cards or even navigation bars.This was made so I could use my reusable code from past projects on new ones if they have any simiarities and it will keep scaling the more projects I work on.
Tech Stack: React , Typescript, TailwindCSS
