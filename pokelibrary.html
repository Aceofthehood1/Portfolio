Pokemon Library

Overview

This project was done during my internship at Turntabl as I was put in a group with another intern to make an angular project using the data from a pokemon api. I worked on the front-end and my partner worked on the backend of the project. The app allows the user to view the details of pokemon shown by taking the pokemon data from an api and representing the data onto the page based on the users selction.

Made with: Angular, Pokemon API
